# Changelog

## v0.12 - Recommendation Engine

- Added account recommendations.
- Added rule-based recommendation evaluation.
- Added recommendation dismissal.
- Added recommendation APIs and UI.

## v0.11 - Health Engine

- Added `account_health`.
- Added rule-based health scoring.
- Added Health page and Account Detail health card.

## v0.10 - Behavior Library

- Added behavior templates.
- Added built-in templates: Warm Up, Quick Upvote, Casual Reader, Deep Reader.
- Added campaign template application.

## v0.9 - Behavior Engine

- Added workflow actions: WAIT, SCROLL, OPEN_POST, BACK.
- Added BehaviorService.

## v0.8 - Workflow Engine

- Added `workflow_steps`.
- Added campaign workflows and workflow execution.

## v0.7 - Campaign Engine

- Added campaigns and campaign account assignments.
- Added campaign run orchestration through existing UpvoteService.

## v0.6 - Upvote Engine

- Added multi-account sequential upvote execution.
- Added Reddit button detection and click verification best effort.

## v0.5 - Activity Engine

- Added activity audit log.
- Integrated activity records into session and profile operations.

## v0.4 - Profile Sync

- Added Reddit profile sync through persistent browser profiles.
- Added profile fields on accounts.

## v0.3 - Browser Session Engine

- Added provider registry.
- Added RedditSessionProvider.
- Added persistent Playwright browser profiles.
- Added session create, finish, validate, refresh, delete, open browser, and open home.

## v0.2 - Account Detail Workspace

- Added account detail route and workspace tabs.
- Added session UI.

## v0.1 - Account Engine

- Added account CRUD.
- Added backend FastAPI foundation.
- Added frontend dashboard and account management foundation.
